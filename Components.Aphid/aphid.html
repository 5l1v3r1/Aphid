Library\Meta.alx Functions


frame(index)
Summary
Retrieves the frame currently on the call stack at the specified index, with the current call. at zero
Parameters
index The index of the frame to be retrieved.
Returns
An AphidFrames representative of a call on the stack.
Example
#'std';
frame(1) |> dump;

frames()
Summary
Retrieves the frames currently on the call stack, including the current call.
Returns
A collection of AphidFrames representative of the current call stack.
Example
#'std';
frames() |> dump;

mutate(mutateNode, rootNode)
Summary
Takes an abstract syntax tree and performs transformations on it.
Parameters
mutateNode The function to be applied to each node as the tree is walked.
rootNode The root at which the transformation begins. Can be a single node or a list of nodes.
Returns
A transformed copy of an abstract syntax tree.
Example
var f = @() print(foo\bar\test.txt);

var block = f.Body @mutate(@{
    if ('\\' |> getToken |> expression.isOp) {
        hasChanged = true;
        ret expression
            .ToString()
            .Replace('\\', '\\\\')
            @format('"{0}"')
            |> ast.str;
    }
});

block |> evalAst;

tokenize(text)
Summary
Tokenizes text using the Aphid lexical analyzer.
Parameters
text The text to be passed to the Aphid lexer.
Returns
An array of AphidTokens.
Example
#'std';
'var x = 10; x *= 2;' |> tokenize |> print;
